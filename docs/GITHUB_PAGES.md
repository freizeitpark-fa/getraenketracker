# GitHub Pages Deployment

## Dateien ersetzen

1. ZIP entpacken.
2. Inhalt des Ordners `CruiseSip/` vollständig markieren.
3. In das lokale GitHub-Repository kopieren.
4. Vorhandene Dateien ersetzen.
5. GitHub Desktop öffnen.
6. Änderungen prüfen.
7. Commit erstellen.
8. Push durchführen.

## GitHub Pages prüfen

Nach dem Push die GitHub-Pages-Seite einmal im Browser öffnen. Der Service Worker speichert die Dateien offline.

## iPhone prüfen

1. Safari öffnen.
2. GitHub-Pages-Link öffnen.
3. Teilen → Zum Home-Bildschirm.
4. App über das Icon starten.
5. Onboarding abschließen.
6. Flugmodus aktivieren und App erneut öffnen.
